AUTOMATION APPROACH
===================
- Playwright (Python) for browser automation of ScriptCase IDE
- PHP scripts for direct SQLite DB manipulation (Attr fields, events, lookups)
- ScriptCase source code is SourceGuardian encrypted - cannot call internal APIs
- Attr1-8 fields in sc_tbapl contain 300+ serialized PHP array keys per field
- Direct DB injection works for: lookups (Def_Complemento), events (sc_tbevt), menu (navmenu_itens JSON)
- Direct DB injection fails for: new app creation (needs full Attr initialization by SC generator)
- Express/Batch creation in IDE is the only reliable way to create new apps
